Reconfigurable Virtual Collective Trainer https://ste-rvct.com

Description

RVCT is a program developed to fulfill a contract under the U.S. Department of Defense. It provides the capability for military personnel to train in a virtual, collective environment. The hardware stations are assembled of reconfigurable kits to simulate a variety of real-world military vehicles including ground and air.

The website acts as a portal which provides links to related developer tools such as Jira, Bitbucket and a scheduling system which allows developers to schedule time to use shared resources. For administrative users, dashboards are provided to aid in program management.

There are plans for more public features such as a news feed, and a point-of-need calculator. These will allow base commanders across the country to determine what steps are required to conduct a training event at their location.

Tech Stack

Frontend
Angular using Material Design.
Backend
NestJS (a Typescript-based framework) with a Postgres database for application data, and an LDAP database for user data.
Deployment
Frontend and backend applications are hosted in Docker containers on RHEL8 Linux servers. The application and database servers are air-gapped for security, and an Nginx reverse proxy is set up on a DMZ server to handle traffic. A separate SMTP relay exists for sending emails.

 Key Terrain https://keyterrain.io

Description

Key Terrain was conceived due to a recognized need in the training simulation industry. Many training simulation programs have a contractual requirement to meet realism requirements. One of the most difficult parts of meeting this requirement is providing terrain, trees, buildings, and infrastructure that accurately reflects the true on-the-ground reality.

An R&D effort at Cole Engineering began in order to generate terrain files in various formats. A web interface was determined to be the most convenient way for customers to select their area of interest and place orders. This became the first e-commerce product developed by Cole Engineering Services.

Tech Stack

Frontend
Angular using Bootstrap and JHipster (a highly-opinionized framework that integrates frontend, backend, and deployment).
Backend
Spring Boot with a Postgres database (provided by the JHipster framework).
Deployment
AWS's ElasticBeanstalk was used as the hosting environment. Deployment was achieved with using JHipster CLI commands after AWS CLI configuration.

Dakota's Impact

  • Developed database models.
  • Designed and implemented web features such as an interactive map, invoicing, order history, and administrative dashboard.
  • Integrated third-party e-commerce services such as Stripe and TaxJar.
  • Planned and managed tasks using Jira.
 Training Readiness Portal https://training-portal-app.shastaworks.com

Description

This training readiness portal was never fully developed, but was meant to be a prototype to support a contract proposal. The potential customer was an organization that provided training services to multiple companies in various subjects. They required a logistics management solution to assist in coordinating class times, registration, trainer availability, and resource availability. Their current management methods involved sharing Excel documents via email. It seemed that a website would be cleaner, faster, and more reliable.

Tech Stack

Frontend
Angular
Backend
None required for this mock up.
Deployment
Slight modifications were made to Angular routing in order to allow demoing the application from the local file system.
